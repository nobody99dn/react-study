## GET STARTED

- Clone project:

  ```bash
  $ git clone git@gitlab.asoft-python.com:van.tran/react-training.git
  ```

- Checkout branch:

  ```bash
  $ git checkout feature/storybook-practice
  ```

- Change directory:

  ```bash
  $ cd room-time-webapp
  ```

- Install packages:

  ```bash
  $ npm install
  ```

- Start project

  ```bash
  $ npm start
  ```

- Open browser and and type http://localhost:1234 in address bar
