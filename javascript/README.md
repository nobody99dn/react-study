# JavaScript training

**_This is my javascript training place_**

## Practice: TODO APP

### GET STARTED

- Clone project:

  ```bash
  $ git clone https://github.com/vawn-t/code-studies.git
  ```

- Checkout branch:

  ```bash
  $ cd javascript
  ```

- Change directory:

  ```bash
  $ cd todos-app
  ```

- Install packages:

  ```bash
  $ npm install
  ```

- Start project

  ```bash
  $ npm start
  ```

- Open browser and and type http://localhost:1234 in address bar
