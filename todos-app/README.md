## GET STARTED

- Clone project:

  ```bash
  $ git clone git@gitlab.asoft-python.com:van.tran/javascript-training.git
  ```

- Checkout branch:

  ```bash
  $ git checkout practice/todo-app
  ```

- Change directory:

  ```bash
  $ cd todos-app
  ```

- Install packages:

  ```bash
  $ npm install
  ```

- Start project

  ```bash
  $ npm start
  ```

- Open browser and and type http://localhost:1234 in address bar
