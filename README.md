## Practice 1: Room Time Web Application (4 days)

### Usage

- Clone project: `git clone git@gitlab.asoft-python.com:van.tran/react-training.git`
- `git checkout develop`
- `cd room-time-webapp`
- run `npm i` then run `npm start`
